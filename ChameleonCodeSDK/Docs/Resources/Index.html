Google Resource Lists

There are 4 lists of resources defined in the google resources folder in this project. They are.

Descriptions

    Properties

    1. View Edit
      AppUser
      A user within the app
    2. View Edit
      Puzzle
      A clue, or puzzle in a game
    3. View Edit
      TreasureHunt
      A game with a specific start date, and list of "clues"
    4. View Edit
      CharacterTemplate
      A character (with a particular vice)

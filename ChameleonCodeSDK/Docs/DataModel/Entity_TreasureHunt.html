Treasure Hunt

A game with a specific start date, and list of "clues"

Edit Attributes

    5 Suggested Attributes

    The following items are in the spreadsheet, but are not yet part of the formal model.
    1. Name
      A name for the hunt

    2. City
      A city that the hunt takes place in.

    3. State
      A state where the hunt takes place

    4. StartDate
      Date that the hunt starts

    5. Prize
      A description for the price

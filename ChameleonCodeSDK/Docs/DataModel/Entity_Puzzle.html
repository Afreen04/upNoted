    7 Suggested Attributes

      The hunt associated

    2. Name
      a puzzle

    3. Hint
      Clue to location

    4. Photo
      Photo of location

    5. Question
      Verification question

    6. Answer
      Correct response

    7. Treat
      Reward for correct answer
